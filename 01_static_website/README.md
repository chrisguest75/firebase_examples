# README

Demonstrates a simple static website on `firebase`  

## Prerequisites

```sh
brew info firebase-cli

curl -sL https://firebase.tools | bash

nvm use

firebase login
```

## Creating a project

```sh
firebase projects:list

firebase init hosting

> Project Name - cg-01-static-website-001

firebase deploy
```
